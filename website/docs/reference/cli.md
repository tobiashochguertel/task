# CLI Usage

Usage: `task [flags] [tasks]...`

## Flags

*   `-l`, `--list`: List tasks
*   `-w`, `--watch`: Watch for changes
*   `-f`, `--force`: Force execution of tasks
*   `-s`, `--silent`: Silent mode
*   `-d`, `--dir`: Sets the working directory
*   `-t`, `--taskfile`: Sets the Taskfile to use

## Completion

Task supports shell completion for Bash, Zsh, Fish, and PowerShell.
