# Core Concepts\n\n## What is a Task?\nA task is a named collection of commands that you want to run together. It's the fundamental building block of a Taskfile.\n\n## How Tasks Relate\nTasks can depend on other tasks, call other tasks, and run in specific orders.\n\n## Variables in Task\nVariables allow you to make your tasks dynamic and reusable.\n\n## Running Tasks\nYou run tasks using the `task` CLI command.
