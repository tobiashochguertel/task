# Taskfile Property Reference

Complete reference for all Taskfile properties.

## All Properties

| Property | Type | Description |
|----------|------|-------------|
| [aliases](./properties/aliases.md) | `array` | Alternative names for the task. |
| [cmds](./properties/cmds.md) | `cmds` | Commands to execute for this task. |
| [deps](./properties/deps.md) | `deps` | Task dependencies that run before this task. |
| [desc](./properties/desc.md) | `string` | Short description shown in task list. |
| [dir](./properties/dir.md) | `string` | Working directory for task execution. |
| [dotenv](./properties/dotenv.md) | `array` | Load environment variables from .env files. |
| [env](./properties/env.md) | `env` | Environment variables for task execution. |
| [generates](./properties/generates.md) | `array` | Output files generated by this task. |
| [method](./properties/method.md) | `string` | Method for checking if task is up-to-date. |
| [platforms](./properties/platforms.md) | `platforms` | Operating systems where task can run. |
| [preconditions](./properties/preconditions.md) | `array` | Conditions that must be met before task runs. |
| [prompt](./properties/prompt.md) | `union` | Show confirmation prompt before executing. |
| [requires](./properties/requires.md) | `requires_obj` | Require specific Task variables to be set. |
| [run](./properties/run.md) | `run` | Control when task should run. |
| [set](./properties/set.md) | `array` | Enable POSIX shell options. |
| [shopt](./properties/shopt.md) | `array` | Enable bash shell options. |
| [silent](./properties/silent.md) | `boolean` | Suppress command output. |
| [sources](./properties/sources.md) | `array` | Source files to check for staleness. |
| [status](./properties/status.md) | `array` | Shell commands to check if task is up-to-date. |
| [summary](./properties/summary.md) | `string` | Long description shown with task --summary. |
| [vars](./properties/vars.md) | `vars` | Global variables available to all tasks. |
| [watch](./properties/watch.md) | `boolean` | Enable watch mode for continuous execution. |
