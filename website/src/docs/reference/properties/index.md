# Taskfile Property Reference

Complete reference for all Taskfile properties.

## All Properties

| Property | Type | Description |
|----------|------|-------------|
| [aliases](./aliases.md) | `array` | Alternative names for the task. |
| [cmds](./cmds.md) | `cmds` | Commands to execute for this task. |
| [deps](./deps.md) | `deps` | Task dependencies that run before this task. |
| [desc](./desc.md) | `string` | Short description shown in task list. |
| [dir](./dir.md) | `string` | Working directory for task execution. |
| [dotenv](./dotenv.md) | `array` | Load environment variables from .env files. |
| [env](./env.md) | `env` | Environment variables for task execution. |
| [generates](./generates.md) | `array` | Output files generated by this task. |
| [includes](./includes.md) | `map[string]Include` | Include external Taskfiles. |
| [interval](./interval.md) | `string` | Interval for watch mode polling. |
| [method](./method.md) | `string` | Method for checking if task is up-to-date. |
| [output](./output.md) | `string` | Control how task output is displayed. |
| [platforms](./platforms.md) | `platforms` | Operating systems where task can run. |
| [preconditions](./preconditions.md) | `array` | Conditions that must be met before task runs. |
| [prompt](./prompt.md) | `union` | Show confirmation prompt before executing. |
| [requires](./requires.md) | `requires_obj` | Require specific Task variables to be set. |
| [run](./run.md) | `run` | Control when task should run. |
| [set](./set.md) | `array` | Enable POSIX shell options. |
| [shopt](./shopt.md) | `array` | Enable bash shell options. |
| [silent](./silent.md) | `boolean` | Suppress command echo before execution. |
| [sources](./sources.md) | `array` | Source files to check for staleness. |
| [status](./status.md) | `array` | Shell commands to check if task is up-to-date. |
| [summary](./summary.md) | `string` | Long description shown with task --summary. |
| [vars](./vars.md) | `vars` | Variables for tasks and templates. |
| [version](./version.md) | `string` | Taskfile schema version. |
| [watch](./watch.md) | `boolean` | Enable watch mode for continuous execution. |
