# generates

Output files generated by this task.

## Type

`array`


## Description

List of files or glob patterns for generated outputs. Used with sources to check if task is up-to-date.




## Contexts

This property can be used in:


- Task level








## Examples


### Basic generates

Track generated files

```yaml
tasks:
  build:
    sources:
      - src/**/*.go
    generates:
      - bin/app
    cmds:
      - go build -o bin/app

```



### Multiple outputs

Track multiple generated files

```yaml
tasks:
  assets:
    sources:
      - assets/icons/*.svg
    generates:
      - dist/icons/*.png
      - dist/sprite.svg
    cmds:
      - npm run build-assets

```
