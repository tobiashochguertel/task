# generates

Output files generated by this task.

## Type

`array`


## Description

Glob patterns for output files produced by the task. Used together with sources to enable Task's incremental build feature. Task compares source and generated file checksums (or timestamps) to determine if task execution is necessary.

When both sources and generates are specified, Task will skip execution if all source files are older than (or have same checksum as when) generated files were created.




## Contexts

This property can be used in:


- Task level








## Examples


### Single output file

Task generates one file

```yaml
tasks:
  build:
    sources:
      - src/**/*.go
    generates:
      - bin/app
    cmds:
      - go build -o bin/app

```



### Multiple output files

Task generates several files

```yaml
tasks:
  bundle:
    sources:
      - src/**/*.js
      - webpack.config.js
    generates:
      - dist/bundle.js
      - dist/bundle.js.map
      - dist/vendor.js
    cmds:
      - webpack --mode production

```



### Glob pattern outputs

Generate files matching pattern

```yaml
tasks:
  compile-ts:
    sources:
      - src/**/*.ts
    generates:
      - dist/**/*.js
      - dist/**/*.d.ts
    cmds:
      - tsc

```



### CSS processing

Compile and minify styles

```yaml
tasks:
  css:
    sources:
      - src/css/**/*.scss
    generates:
      - public/styles/main.css
      - public/styles/main.css.map
    cmds:
      - sass --source-map src/css/main.scss public/styles/main.css
      - postcss public/styles/main.css --replace

```
